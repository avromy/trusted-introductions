# Design System

## Status

Not implemented. This document defines initial direction.

## Tone

Warm, trustworthy, calm, and community-oriented. Avoid growth-hacking, sales, or recruiting-agency language.

## Visual Principles

- High readability.
- Clear hierarchy.
- Accessible contrast.
- Minimal decorative UI.
- Strong empty states explaining trust and privacy.

## Component Priorities

- Invite card.
- Onboarding stepper.
- Profile privacy controls.
- Match explanation card.
- Introduction workflow timeline.
- Outcome capture form.
