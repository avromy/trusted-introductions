# Roadmap

## Phase 0: Repository Operating System

Create docs, ADRs, templates, and CI baseline.

## Phase 1: Application Foundation

Select stack, scaffold app, configure auth, database, CI, deployment, and observability.

## Phase 2: Invite and Identity

Implement invite-only onboarding, one-person identity model, profile creation, privacy settings, and steward controls.

## Phase 3: Matching and Introductions

Implement helper capabilities, seeker requests, explainable match suggestions, manual steward review, and introduction workflows.

## Phase 4: Outcomes and Community Health

Implement outcome tracking, follow-ups, dashboards, and quality monitoring.

## Phase 5: Controlled Expansion

Add public meet pages, advanced feature flags, improved matching, and carefully scoped AI assistance.
